# **Система автоматизации учета оборудования ВГУ**

**Веб-приложение для централизованного управления оборудованием Воронежского государственного университета.**  
Позволяет вести инвентаризацию, генерировать QR-коды и визуализировать расположение техники на интерактивной карте.

**Стек:** React + TypeScript, FastAPI (Python), PostgreSQL.
